Course Details

Computer Science and Engineering

Overview

The Computer Science and Engineering (CSE) program is designed to provide students with a strong foundation in computing principles, software development, and emerging technologies. The course integrates theoretical knowledge with practical skills to prepare students for successful careers in the IT and software industry.

Program Objectives

  1. To build strong fundamentals in programming, algorithms, and data structures
  2. To develop problem-solving and analytical thinking skills
  3. To expose students to modern technologies such as AI, ML, Data Science, and Cloud Computing
  4. To prepare graduates for industry, higher studies, research, and entrepreneurship

Core Subjects

  • Programming in C / Python / Java
  • Data Structures and Algorithms
  • Object-Oriented Programming
  • Database Management Systems
  • Operating Systems
  • Computer Networks
  • Software Engineering
  • Web Technologies
  • Artificial Intelligence and Machine Learning

Industry Relevance

The CSE curriculum is continuously updated to match industry trends, ensuring students gain relevant skills in cutting-edge technologies and are ready for real-world challenges.

Cyber Security

Overview

The Cyber Security program focuses on protecting computer systems, networks, and data from cyber threats and attacks. The course equips students with the knowledge and skills required to design secure systems and implement effective security solutions.

Program Objectives

  1. To develop strong fundamentals in information and network security
  2. To train students in ethical hacking and cyber defense techniques
  3. To understand cyber laws, risk management, and security policies
  4. To prepare students for careers in cyber forensics and incident response

Core Subjects

  • Fundamentals of Cyber Security
  • Cryptography and Network Security
  • Ethical Hacking
  • Computer Networks
  • Operating Systems Security
  • Cyber Forensics
  • Web and Application Security
  • Cloud Security
  • Cyber Laws and Compliance

Industry Relevance

The Cyber Security curriculum is aligned with global standards, enabling graduates to handle real-world cyber threats effectively.

Artificial Intelligence and Data Science (AI & DS)

Overview

The AI & DS program focuses on developing intelligent systems and extracting meaningful insights from large datasets. The course combines computer science, statistics, and machine learning for data-driven solutions.

Program Objectives

  1. To build foundations in AI, ML, and Data Analytics
  2. To develop skills in data processing and visualization
  3. To solve real-world problems using intelligent algorithms
  4. To prepare graduates for careers in AI, Data Science, and research

Core Subjects

  • Programming in Python and R
  • Data Structures and Algorithms
  • Probability and Statistics
  • Machine Learning
  • Deep Learning
  • Artificial Intelligence
  • Data Mining and Warehousing
  • Big Data Analytics
  • Natural Language Processing

Industry Relevance

The AI & DS curriculum is designed as per current industry needs, enabling students to work in domains such as healthcare, finance, smart cities, and e-commerce.
